atten_terrestrial¶

pycraf.atm.atten_terrestrial(specific_atten, path_length)[source] [edit on github]¶

Total path attenuation for a path close to the ground (i.e., one layer), according to ITU-R P.676-11, Annex 1 + 2.

Parameters
specific_attenQuantity

Specific attenuation (dry + wet) [dB / km]

path_lengthQuantity

Length of path [km]

Returns
total_attenQuantity

Total attenuation along path [dB]
