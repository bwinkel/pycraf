ptx_from_powerflux¶

pycraf.conversions.ptx_from_powerflux(powerflux, dist, gtx)[source] [edit on github]¶

Transmitter power from power flux density.

Parameters
powerfluxQuantity

Power flux density, S (at distance) [W/m**2, dB_W_m2]

distQuantity

Distance to transmitter [m]

gtxQuantity

Gain of transmitter, Gtx [dBi, dimless]

Returns
ptxQuantity

Transmitter power, Ptx [W]

Notes

Because all parameters/returned values are Astropy Quantities (see Quantity), unit conversion is automatically performed.
