eff_area_from_gain¶

pycraf.conversions.eff_area_from_gain(gain, freq)[source] [edit on github]¶

Effective antenna area from antenna gain, given frequency.

Parameters:

gain : Quantity

Antenna gain [dBi, dimless]

freq : Quantity

Frequency [Hz]
Returns:

eff_area : Quantity

Effective antenna area, A_eff [m**2]

Notes

Because all parameters/returned values are Astropy Quantities (see Quantity), unit conversion is automatically performed.
