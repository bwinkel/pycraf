efield_equivalency¶

pycraf.conversions.efield_equivalency()[source] [edit on github]¶

units equivalency to handle log-scale E-field units.

For electric fields, the Decibel scale is define via the amplitude of the field squared, \({\vert\vec E\vert}^2\) which is proportional to the power.

Returns
equivalencylist

The returned list contains one tuple with the equivalency.
