efield_from_ptx¶

pycraf.conversions.efield_from_ptx(ptx, dist, gtx)[source] [edit on github]¶

E-field strength measured at distance from transmitter power.

Parameters
ptxQuantity

Transmitter power, Ptx [dB_W, W]

distQuantity

Distance to transmitter [m]

gtxQuantity

Gain of transmitter, Gtx [dBi, dimless]

Returns
efieldQuantity

E-field strength, E [uV/m]

Notes

Because all parameters/returned values are Astropy Quantities (see Quantity), unit conversion is automatically performed.
