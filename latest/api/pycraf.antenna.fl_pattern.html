fl_pattern¶

pycraf.antenna.fl_pattern(phi, diameter, wavelength, G_max)[source] [edit on github]¶

Antenna gain as a function of angular distance after ITU-R Rec F.699.

Parameters
phiQuantity

Angular distance[deg]

diameterQuantity

Antenna diameter [m]

wavelengthQuantity

Observing wavelength [m]

G_maxQuantity

Antenna maximum gain [dBi]

Returns
gainQuantity

Antenna gain [dBi]

Notes

  • If G_max is unknown, there is a helper function pycraf.antenna.G_max_from_HPBW that can be used to estimate its value.

  • See ITU-R Rec F.699 for further explanation and applicability of this model.
