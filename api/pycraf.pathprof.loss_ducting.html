loss_ducting¶

pycraf.pathprof.loss_ducting(pathprop)[source]¶

Calculate the ducting/layer reflection loss, L_ba, of a propagating radio wave according to ITU-R P.452-16 Eq (46-56).

Parameters:

pathprop : PathProp instance

This helper class works as a container to hold various properties of the path (e.g., geometry).
Returns:

L_ba : Quantity

Ducting/layer reflection loss [dB]
