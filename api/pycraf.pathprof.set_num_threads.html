set_num_threads¶

pycraf.pathprof.set_num_threads(int nthreads)¶

Change maximum number of threads to use.

Parameters
nthreads - int

Number of threads to use.

Notes

  • This can also be controlled by setting the environment variable OMP_NUM_THREADS.
